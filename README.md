PNU_Landmark_Classification
============================

## ABOUT
이 project는 부산대학교 명소 5곳을 Supervised Learning시킨 결과물이다.
명소 5곳은 '웅비의 탑', '대학 본부', '5층 석탑', '제도관', '중앙 도서관'이다.

## VERSION
- tensorflow 2.2.0
- keras 2.3.1
- python 3.8
